Tutorial Deploy GitHub

Berikut merupakan langkah-langkah yang harus dilakukan untuk deploy project ke GitHub

Pastikan :

  1. Memiliki akun GitHub.
  2. Install git pada komputer/laptop.
  3. Siapkan project yang akan di deploy.

Langkah-langkah untuk Deploy :

  1. Buat Repository Baru
    • Masuk ke akun GitHub.
    • Kemudian klik ikon + di pojok kanan atas dan pilih New repository.
    • Setelah itu isi nama repository pada bagian Repository name.
    • Tulis deskripsi project (opsional).
    • Pilih Public agar semua orang dapat mengakses repositori atau pilih Private sehingga orang yang dipilih saja yang dapat mengakses, lalu klik Create repository.
  2. Inisialisasi Git di Direktori Proyek Anda 
    cd direktori-nama_project
git init
  3. Tambahkan File 
    git add .
  4. Commit Perubahan 
    git commit -m "first commit "
  5. Hubungkan Repository Lokal ke GitHub 
    git remote add origin https://github.com/username/nama-repository.git
  6. Kirim Kode ke GitHub 
    git push -u origin main

Kesimpulan

Project sekarang telah berhasil di-deploy ke GitHub. Untuk memastikan bahwa semua file telah diunggah dengan benar bisa dengan membuka URL repository GitHub yang telah dibuat. Di sana terdapat semua file project beserta riwayat commit. URL ini juga dapat dibagikan kepada orang lain apabila mereka mengakses atau berkolaborasi dalam proyek tersebut. Pastikan repository diatur ke publik jika ingin membagikannya kepada pengguna umum atau ke private jika hanya pengguna tertentu yang boleh mengaksesnya.


Tutorial Membuat Live Link (GitHub Pages)

Berikut merupakan langkah-langkah yang harus dilakukan untuk membuat live link di GitHub

  1. Pastikan project sudah ada di repositori github
  2. Masuk ke halaman repositori, kemudian klik tab settings
  3. Pada halaman Settings gulir ke bawah hingga menemukan bagian Pages
  4. Pada bagian Source, pilih cabang (branch) yang ingin Anda gunakan sebagai sumber GitHub Pages. Pilih main, kemudian pilih direktori root dengan opsi /(root)
  5. Setelah mengatur cabang dan direktori, klik Save untuk menyimpan pengaturan.
  6. GitHub akan memproses pengaturan GitHub Pages. Tunggu beberapa saat hingga tautan GitHub Pages tersedia.
  7. Setelah konfigurasi selesai, GitHub akan menampilkan tautan GitHub Pages di bagian atas pengaturan. Klik tautan tersebut untuk membuka live preview dari project.
